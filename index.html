Introduction

PHP (recursive acronym for PHP: Hypertext Preprocessor) is a widely-used open source general-purpose scripting language that is especially suited for web development and can be embedded into HTML.

The PHP code is delimited by 2 tags (start and end) and it doesn't take many commands to display HTML using PHP, something that happens in languages ​​like Perl and C.

Unlike JavaScript, PHP code is executed by the server, generating HTML which is then sent to the browser.

One of the advantages of using PHP is that it is very easy to use for a beginner but offers a lot of features for professionals.

What PHP can do

PHP is focused on server-side scripting, so you can perform tasks like capturing form data, sending and receiving cookies, generating dynamic content, or anything a CGI program can do. However, PHP can do much more.

Main areas where PHP scripts are used:

  • Server-side scripts. It is the most common area where PHP works. You need 3 things for this to work: a PHP interpreter, a web server and a web browser. You need to run the web server connected to a PHP installation. You can access the results of your PHP program with a web browser by viewing the PHP page through the web server. All of this can run on your home machine if you are just trying out programming with PHP.
  • Command line scripts. You can run scripts with PHP code in the terminal without the need for a web server. The only thing you need is the PHP interpreter installed on your machine. This usage type is ideal for scripts executed using cron (Unix, Linux) or Task Scheduler (on Windows). These scripts can also be used for simple text processing routines.
  • Develop desktop applications. PHP is probably not the best programming language for this, but if you know PHP well and want to use its advanced features to build desktop applications with graphical interfaces, you can use PHP-GTK which is a PHP extension for that, also serving to create cross-platform applications. PHP-GTK is not available in the official distribution. If interested, visit the project's website.

Hello world

Create a file with the extension ".php" and put the following code in it:

<?php echo "Hello World!"; ?>

To run the script, enter the following command in any command-line interpreter:

"php -f myscript.php"

When PHP parses a file it looks for the opening and closing tags, <?php and ?>, which tell PHP to start or stop parsing the code between them. Interpretation like this allows PHP to be included in various types of documents, as everything outside these tags is ignored by the PHP interpreter.

To run the file, just start the web server (Apache) and open the file in the browse.

Variables

You use variables as symbolic names for values in your application. The names of variables, called identifiers, conform to certain rules.

Variables in PHP are represented by a dollar sign ($) followed by the variable name. Variable names are case-sensitive (that is, they are case-sensitive).

Variable names follow the same rules as other labels in PHP. A valid variable name starts with a letter or underscore, followed by any number of letters, numbers, or underscores. In a regular expression, it could be represented like this: '[a-zA-Z_\x7f-\xff][a-zA-Z0-9_\x7f-\xff]*'

<?php $myVar = 'some text'; echo $myVar; $4site = 'not yet'; // not valid; starts with a number $_4site = 'not yet'; // valid; starts with underscore $täyte = 'mansikka'; // valid; 'ä' is an ASCII (extended) character 228 ?>

Function declarations

A function definition (also called a function declaration, or function statement) consists of the function keyword, followed by:

  • The name of the function.
  • A list of arguments to the function, enclosed in parentheses and separated by commas.
  • The PHP statements that define the function, enclosed in curly brackets, { }.
<?php function foo($arg_1, $arg_2, /* ..., */ $arg_n) { echo "Example.\n"; return $value_returned; } ?>
Variable scope

The scope of a variable is the context where it was defined. Most PHP variables have local scope only. This local scope includes included and required files. For example:

<?php $a = 1; /* global scope */ function Test() { echo $a; /* local scope (not defined) */ } Test(); ?>

The below script will print 3. Declaring $a and $b global in the function will cause all references to these variables to reference the global version. There is no limit to the number of global variables that can be manipulated by a function.

A second way to access variables from the global scope is using the special $GLOBALS array defined by PHP.

<?php $a = 1; $b = 2; function Sum() { $GLOBALS['b'] = $GLOBALS['a'] + $GLOBALS['b']; } Sum(); echo $b; ?>
Constants

Constants can be defined using the const keyword, or by using the define()-function. While define() allows a constant to be defined to an arbitrary expression, the const keyword has restrictions as outlined in the next paragraph. Once a constant is defined, it can never be changed or undefined.

When using the const keyword, only scalar (bool, int, float and string) expressions and constant arrays containing only scalar expressions are accepted. It is possible to define constants as a resource, but it should be avoided, as it can cause unexpected results.

These are the differences between constants and variables:

  • Constants do not have a dollar sign ($) before them;
  • Constants may be defined and accessed anywhere without regard to variable scoping rules;
  • Constants may not be redefined or undefined once they have been set; and
  • Constants may only evaluate to scalar values or arrays.

Some examples:

<?php define("CONSTANT", "Hello world."); echo CONSTANT; // outputs "Hello world." echo Constant; // Emits an Error: Undefined constant "Constant" // Prior to PHP 8.0.0, outputs "Constant" and issues a warning. // Scalar expression const ANOTHER_CONST = 'some text'; const ANIMALS = ['dogs', 'cats', 'ducks']; echo ANOTHER_CONST; ?>
If else statement

The if construct is one of the most important features of many languages, PHP included. It allows for conditional execution of code fragments. PHP features an if structure that is similar to that of C.

Some examples:

<?php # if $a = 1; $b = 2; // if if ($a > $b) echo "a is bigger than b"; // if/else $a = false; $b = true; if ($a){ if ($b){ echo "b"; }else{ echo "c"; } } // elseif/else if $a = 100; $b = 300; if ($a > $b) { echo "a is bigger than b"; } elseif ($a == $b) { echo "a is equal to b"; } else { echo "a is smaller than b"; } ?>

While loops are the simplest type of loop in PHP. They behave just like their C counterparts
